
# Система генерации мелодий для пробуждения

Эта система использует Web Audio API для создания приятных мелодий, которые генерируются случайным образом при каждом срабатывании будильника или таймера.

## Структура

- `melody_generator.js` - основной модуль генерации мелодий
- `scales.js` - определения музыкальных гамм и аккордов
- `patterns.js` - паттерны для генерации мелодий
- `player.js` - модуль воспроизведения сгенерированных мелодий

## Использование

Система автоматически интегрирована в основной файл `script.js` и запускается при срабатывании будильника или таймера.
