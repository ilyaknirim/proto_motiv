# Мотивирующий Будильник и Таймер

Современное веб-приложение с будильником и таймером, которое использует мотивирующие аудиофайлы для пробуждения и уведомлений.

## Особенности

- **Будильник**: Установите время и дни недели для срабатывания будильника
- **Таймер**: Настройте обратный отсчет для отслеживания времени
- **Случайные мелодии**: Каждое срабатывание будильника или таймера сопровождается случайной мотивирующей мелодией из коллекции
- **Адаптивный дизайн**: Интерфейс оптимизирован для использования на мобильных устройствах
- **Сохранение настроек**: Ваши настройки будильника сохраняются в браузере

## Использование

### Будильник

1. Перейдите на вкладку "Будильник"
2. Установите желаемое время с помощью полей ввода часов и минут
3. Выберите дни недели, когда должен срабатывать будильник
4. Нажмите кнопку "Установить будильник"
5. Для отмены нажмите кнопку "Отменить будильник"

### Таймер

1. Перейдите на вкладку "Таймер"
2. Установите время для обратного отсчета (часы, минуты, секунды)
3. Нажмите кнопку "Запустить"
4. Для приостановки нажмите "Пауза", для сброса - "Сброс"

## Развертывание на GitHub Pages

1. Форкните этот репозиторий
2. В настройках репозитория перейдите в раздел "Pages"
3. В разделе "Build and deployment" выберите источник "Deploy from a branch"
4. В качестве ветки выберите "main" и папку "/root"
5. Нажмите "Save"
6. Через несколько минут ваш сайт будет доступен по адресу: `https://yourusername.github.io/repository-name`
